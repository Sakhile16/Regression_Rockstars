# Regression_Rockstars
MIT 808 Semester Project

The SemesterProjectPart1.py is the Python file used for EDA visuals, except for the map, which was done in QGIS.
Also, note that the data that will be used in the project can not be uploaded here because of restrictions.
The following are two data sets that will be used in this project:

Drone imagery: A very high-resolution imagery (i.e., 3 cm) of the landscape in Melusi, collected in 
2025 by the Department of Geography, Geoinformatics and Meteorology (GGM) at the University of 
Pretoria. 

GRT-I Baseline 2023: A demographic household survey conducted in 2021, 2022 and 2023 in 
Melusi by the Gauteng Research Triangle.
